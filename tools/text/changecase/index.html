Frequently Asked Questions

Does the tool support developer case formats like camelCase?

Yes. Developer-friendly formats including camelCase, PascalCase, snake_case, and kebab-case are all supported alongside standard prose formats.

Is there a limit to how much text I can convert?

No. The tool handles any amount of text — from a single word to entire documents.

What's the difference between Title Case and Sentence case?

Title Case capitalizes the first letter of every major word (used in headings and titles). Sentence case capitalizes only the first word of each sentence, like standard prose.
