Frequently Asked Questions

Will formatting like bold and tables be preserved?

The tool extracts plain text only. Formatting is intentionally not preserved — plain text is the desired output for most extraction use cases.

Can I extract text from a scanned PDF?

Scanned PDFs are images of text rather than actual text content, so standard extraction won't work on them. The file needs to have been digitally created or run through OCR first.

Is there a document length limit?

No server-side limits apply. Very large documents may be affected by your browser's available memory.
