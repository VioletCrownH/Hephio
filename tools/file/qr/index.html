Frequently Asked Questions

What can I encode in a QR code?

Any text-based content can be encoded: URLs, plain text, email addresses, phone numbers, Wi-Fi credentials, and vCard contact information are all common uses.

Should I download as PNG or SVG?

Use SVG for print and design work — it scales to any size without losing sharpness. Use PNG for web use and applications that don't support SVG.

Will the QR code work forever?

The QR code itself doesn't expire. However, if it encodes a URL, the code stops working if that URL goes offline. For permanent use, encode a URL you control and keep active.
