Frequently Asked Questions

What information does the image info tool show?

The tool displays pixel dimensions (width and height), file size, file format, color space (sRGB, CMYK, etc.), bit depth, and DPI/PPI where the data is available in the file.

Why would I need to check DPI?

DPI (dots per inch) determines print quality. Most print applications require at least 300 DPI for sharp results. Images downloaded from the web are often 72 DPI — fine for screens, but too low for printing.

Is this different from right-clicking "Get Info" on my computer?

The tool may surface additional technical details beyond what your OS shows, including color profile and bit depth information embedded in the file itself.
